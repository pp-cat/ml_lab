pip install ipykernel
python -m ipykernel install --user --name=myenv --display-name "My Virtual Env"
--user: Installs the kernel for the current user.
--name=myenv: Specifies the internal name for the kernel (should match your virtual environment name).
--display-name "My Virtual Env": Sets the name that will appear in the Jupyter Notebook interface.
